# Epic Seven Astral Forge Speed Refresher

This is an app made to automatically refresh astral forge in search of 5 speed. The app autoclicks the refresh button until it sees a 5 speed roll. It takes over your mouse, moving the cursor to the refresh button and auto clicks it.

## Getting Started
**Step 1.** Head over to the [releases page](https://github.com/DFong298/e7-astrium-speed-refresher/releases) and download the .zip file of the latest release (astrium-speed-refresher.zip)

**Step 2.** Extract the zip file to any directory

**Step 3.** Launch Epic Seven, and navigate to the Astral Forge, to the screen where you can start refreshing substats

**Step 4.** Launch "Astrium Speed Refresher.exe" from the folder you just extracted with administrator privileges (very important)

**Step 5.** Press "Start Refresh"

**Step 6.** Touch grass until you run out of astral forge points or hit a 5 speeder

## Running Locally
If you want to use the program locally instead of just using the .exe, you will need to install and configure Tesseract yourself.
